# Contributing to Ina Zaoui's Portfolio

Thank you very much for your interest in this project! The goal of it is to provide Ina place to post
her pictures and promote other photographers.

## Submitting changes

If you want to suggest any change or new feature to the code please send a pull request.

- Fork the repository and create your branch from `main`.
- Add new tests if your new code should be tested.
- Code coverage must remain above **70%**.
- Ensure the test suite passes.
- Issue the pull request.

## Reporting issues

If you find a bug you can report it by opening an issue, make sure you give as many details as possible:

- A description of the bug you encountered
- Steps to reproduce it (give sample code if possible)
- What behaviour was expected
- The actual behaviour
- Fix you already tried etc...

## Contributing documentation

- Keep the `README.md` up to date if any of your changes affects installation or usage.
- Write docblocks for new classes and methods.
- Use English for your code, comments and doc.

## Coding conventions

This project is using the PSR-12 coding style, you can find additional information here:
